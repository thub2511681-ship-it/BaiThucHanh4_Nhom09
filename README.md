BUỔI THỰC HÀNH 4: MICROSOFT POWERPOINT

1. Mục tiêu
   
- Sau khi hoàn thành bài thực hành này, người học có thể:
- Hiểu và sử dụng thành thạo Google Docs để tạo và chỉnh sửa văn bản trực tuyến
- Thực hiện các thao tác định dạng văn bản cơ bản và nâng cao
- Biết cách chia sẻ tài liệu, phân quyền người dùng
- Kiểm tra lịch sử chỉnh sửa (Version History)
- Làm quen với tính năng làm việc nhóm và Meeting trong Google Docs

2. Nội dung thực hành từng phần
   
 2.1. Tạo và quản lý tài liệu Google Docs
   
- Tạo một tài liệu mới trên Google Docs
- Đặt tên file phù hợp với nội dung bài thực hành
- Lưu trữ và quản lý tài liệu trên Google Drive
- Kiến thức đạt được:
+ Cách khởi tạo và quản lý tài liệu trực tuyến
+ Hiểu cơ chế lưu tự động của Google Docs

 2.2. Tạo và định dạng văn bản

- Thiết lập Page Setup (khổ giấy, lề trang, hướng giấy)
- Định dạng Font chữ (kiểu chữ, cỡ chữ, màu sắc)
- Định dạng Paragraph (căn lề, giãn dòng, thụt đầu dòng)
- Kiến thức đạt được:
+ Áp dụng đúng chuẩn trình bày văn bản
+ Tạo văn bản dễ đọc, rõ ràng và chuyên nghiệp

 2.3. Sử dụng Heading và Caption

- Tạo và chỉnh sửa các Heading (Heading 1, Heading 2, …)
- Áp dụng Heading cho các tiêu đề nội dung
- Sử dụng Caption cho hình ảnh hoặc bảng (nếu có)
- Kiến thức đạt được:
+ Biết cách tổ chức nội dung khoa học
+ Chuẩn bị văn bản cho việc tạo mục lục tự động

 2.4. Chia sẻ tài liệu và phân quyền

- Chia sẻ tài liệu cho người khác
- Thiết lập quyền:
+ Viewer (Chỉ xem)
+ Commenter (Nhận xét)
+ Editor (Chỉnh sửa)
- Kiến thức đạt được:
+ Làm việc nhóm hiệu quả trên Google Docs
+ Quản lý quyền truy cập tài liệu an toàn

 2.5. Kiểm tra Version History và Meeting

- Xem lịch sử chỉnh sửa (Version History)
- Khôi phục phiên bản cũ khi cần
- Sử dụng tính năng Meeting / làm việc trực tiếp trong Google Docs
- Kiến thức đạt được:
+ Theo dõi thay đổi tài liệu theo thời gian
+ Hiểu cách phối hợp làm việc nhóm trực tuyến

3. Yêu cầu bài thực hành

- Tạo 01 file Google Docs hoàn chỉnh
- Nội dung được trình bày đúng định dạng, rõ ràng
- Sử dụng Heading hợp lý cho các phần
- File được chia sẻ đúng quyền theo yêu cầu
- Kiểm tra được lịch sử chỉnh sửa của tài liệu

4. Kết quả cần đạt

- Hoàn thành 01 tài liệu Google Docs đúng yêu cầu
- Văn bản có bố cục rõ ràng, định dạng chuẩn

 Nắm vững kỹ năng làm việc nhóm trên Google Docs
